Portifólio

Felipe Corrêa

Com estudos em relação a melhorias de tempo em respostas referente a criação de cotação em produtos, resolvi fazer um portfólio online, aonde o cliente terá acesso com todo auxílio de responsividade e simplicidade, nas buscas pelos produtos necessários. Nele o cliente poderá selecionar o objeto necessário e informar a quantidade necessária para o seu pedido, sabendo o valor do produto no atacado. Com este projeto, reduziremos para 0 o tempo de resposta entre um e-mail enviado solicitando a cotação dos produtos. Funcionará como um e-commerce, cliente seleciona os produtos, quantidade e informa o método de envio, e irá gerar um QR code para empresa, utilizando API do estoque disponível.

With studies regarding time improvements in responses regarding the creation of product quotations, I decided to create an online portfolio, where the customer will have access with all the help of responsiveness and simplicity, in the searches for the necessary products. In it, the customer will be able to select the necessary object and inform the quantity necessary for his order, knowing the value of the product in wholesale. With this project, we will reduce to 0 the response time between an email sent requesting a quotation for the products. It will work as an e-commerce, the customer selects the products, quantity and informs the shipping method, and will generate a QR code for the company, using the available stock API.
